# Prime Video Movies & TV Shows Analysis Dashboard

## Overview

This project is an interactive Power BI dashboard built using the Prime Video Movies & TV Shows dataset. It provides insights into the platform's content library by analyzing movies and TV shows across multiple dimensions, including genres, ratings, countries, directors, cast, and release trends.

The dashboard is designed to help users explore the dataset through interactive visualizations and gain meaningful insights from the available content.

## Features

- Overview of total titles, content types, countries, ratings, directors, and cast
- Movies vs. TV Shows comparison
- Genre-wise content distribution
- Country-wise content distribution using map visualization
- Ratings distribution across all titles
- Release year trend analysis
- Interactive dashboard with filters and slicers

---

## Dashboard Insights

- Prime Video hosts **9,655** titles.
- The content library spans **87** countries.
- Movies account for approximately **81%** of the available content, while TV Shows make up about **19%**.
- Drama is the most common genre, followed by Comedy and Documentary.
- Content availability has grown significantly in recent years.
